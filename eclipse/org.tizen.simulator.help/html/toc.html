Tizen Simulator

The Tizen Simulator is a light-weight mobile application development and testing tool for Tizen web application developers. It includes the following features:
  • Support for modern HTML5 web standards
  • Simulate Tizen APIs
  • Run and debug Tizen applications started from either the Eclipse-based SDK or loaded directly from the file system
  • Run on Windows, Linux and Mac OS X platforms. (The Eclipse-based SDK is not yet supported on OS X)
For detailed feature descriptions with screenshots, see the Simulator Features topic. The Tizen Simulator is based on the Ripple-Framework. It is licensed under Apache Software License v.2.0.
